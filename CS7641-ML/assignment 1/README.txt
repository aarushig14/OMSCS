Datasets -
------------------------

1. Tic Tac Toe Endgames - https://archive.ics.uci.edu/ml/datasets/Tic-Tac-Toe+Endgame
2. Wine Quality Dataset - https://archive.ics.uci.edu/ml/datasets/wine+quality

Implemented using python 3.9 - sklearn, matplotlib, numpy, pandas. Use pip to install any missing libraries.

Two python code files -
-------------------------

1. analysis.py - used to generate all the individual graphs for individual algorithms.
2. comparison.py - used to generate table for comparison amond algorithms with tuning of hyper-parameters.

How to run python files -
--------------------------

python analysis.py <filename>
python comparison.py <filename>

replace <filename> with the path of dataset.

How results are stored -
--------------------------

Outputs are stored for one dataset at one time in 'results' folder created.
Note: Running another dataset will replace the content of 'results' folder.
